## 1. Model Overview

The training pipeline of our proposed model is shown below:

<div align="center">
  <img src="https://raw.githubusercontent.com/papercodeHua/DAPF_LDM/main/result/exp_1/fig1.png" width="95%">
</div>

MRI-to-PET synthesis often fails to capture disease-specific hypometabolism patterns, limiting its clinical utility for Alzheimer's disease (AD) diagnosis. To address this, we propose **DAPF-LDM**, a Diagnosis-Aware Latent Diffusion Model.

Operating in a compressed 3D latent space, our framework introduces a **Diagnosis-Aware Pathology-Focused (DAPF) block**. This block uses orthogonal decomposition to disentangle anatomical cues from noise and integrates anatomical priors with MMD-based adaptive weighting to explicitly emphasize AD-critical regions. Additionally, to bridge the semantic gap between modalities, we introduce a Cross-Modal Fusion-Perceptual Loss.

## 2. Setup and Dependencies

### Dataset

To train or test the model, you must first download the **ADNI** dataset from [here](https://adni.loni.usc.edu/data-samples/access-data/).

### Dependencies

Requirements: Python 3.8+, PyTorch 1.12+ (CUDA recommended).

```bash
pip install -r requirements.txt
```

## 3. Running the Model

Configurations (batch size, learning rate, paths, etc.) can be modified in `utils/config.py`. The training process consists of two main stages.

### 3.0 Preprocessing

Generate ROI binary masks from the Harvard-Oxford atlas:

```bash
python utils/split_atlas.py --input ./datas/HarvardOxford-sub-maxprob-thr0-1mm_aligned.nii.gz --output ./datas/atlas_masks/
```

### 3.1 Stage I: Auxiliary Component Pre-training

Pre-train the auxiliary components before the main diffusion model.

1. Train 3D AAE (latent representation)

   ```bash
   # Set phase="train" in utils/config.py
   torchrun --nproc_per_node=2 train_aae.py  # Adjust GPU count as needed
   ```

2. Encode PET data to latent space

   ```bash
   # Set phase="encoding" in utils/config.py
   torchrun --nproc_per_node=2 train_aae.py
   ```

3. Train Diagnosis-Aware Network (DA-Net)

   ```bash
   torchrun --nproc_per_node=2 train_da_net.py
   ```

4. Train Perceptual Loss Network (MultiModal ResCNN)

   ```bash
   torchrun --nproc_per_node=2 train_mulrescnn.py
   ```

### 3.2 Stage II: DAPF-LDM Training

Train the main latent diffusion model (with DAPF block and Fusion Loss):

```bash
torchrun --nproc_per_node=2 train_dapf_ldm.py  # Recommended: 2+ GPUs
```

### 3.3 Inference & Evaluation

Generate synthetic PET images on the test set and compute metrics (global: PSNR/SSIM; lesion-specific: ROI-MAE):

```bash
torchrun --nproc_per_node=1 test_dapf_ldm.py
```

Results will be saved as CSV files in the corresponding experiment folder (e.g., `result/exp_2/test.csv`).
