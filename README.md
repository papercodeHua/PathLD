## 1. Summary of the Model

The following shows the training procedure for our proposed model:

<div align="center">
  <img src="assets/framework.png" width="95%">
</div>

MRI-to-PET synthesis often struggles to capture disease-specific hypometabolism patterns, thereby limiting its clinical utility for Alzheimer's disease (AD) diagnosis. To address this, we propose **DAPF-LDM**, a Diagnosis-Aware Latent Diffusion Model.

Operating within a compressed 3D latent space, our framework specifically introduces a **Diagnosis-Aware Pathology-Focused (DAPF) block**. This block employs orthogonal decomposition to disentangle anatomical cues from noise and integrates anatomical priors with MMD-based adaptive weighting to explicitly emphasize AD-critical regions. Furthermore, to bridge the semantic gap between modalities, we propose a Cross-Modal Fusion-Perceptual Loss.

## 3. Running the model
Users can modify the settings in utils/config.py to specify configurations (batch size, learning rate, paths). The training consists of two main stages.
### 3.0 Preprocessing
Generate ROI binary masks from the atlas file:
python tools/split_atlas.py --input ./datas/HarvardOxford-sub-maxprob-thr0-1mm_aligned.nii.gz --output ./datas/atlas_masks/

### 3.1 Stage I: Component Pre-training
Before training the diffusion model, auxiliary components must be trained.
1. Train 3D AAE (Latent Representation)
code
Bash
# Ensure phase="train" in utils/config.py
python train_aae.py
2. Encode PET Data to Latent Space
code
Bash
# Ensure phase="encoding" in utils/config.py
python train_aae.py
3. Train Diagnosis-Aware Network (DA-Net)
code
Bash
# Uses DDP
torchrun --nproc_per_node=2 train_da_net.py
4. Train Perceptual Loss Network (MultiModal ResCNN)
code
Bash
# Uses DDP
torchrun --nproc_per_node=2 train_mulrescnn.py
3.2 Stage II: DAPF-LDM Training
Train the main latent diffusion model with the DAPF block and Fusion Loss:
code
Bash
# Uses DDP (Recommended: 2 GPUs)
torchrun --nproc_per_node=2 train_dapf_ldm.py
3.3 Inference & Evaluation
To generate synthetic PET images from the test set and calculate Global (PSNR/SSIM) and Lesion-specific (ROI-MAE) metrics:
code
Bash
torchrun --nproc_per_node=1 test_dapf_ldm.py
Results will be saved to result/exp_6/test2.csv.
